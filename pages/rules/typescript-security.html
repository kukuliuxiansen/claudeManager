TypeScript 安全规范

定义类型安全边界、输入验证、XSS防护和依赖安全检查

类型: 语言规则

实现效果

TypeScript 安全规范确保类型安全边界和完整的安全防护。

  • 类型安全边界 - 定义可信任与不可信任数据的边界
  • 输入验证 - 使用 Zod 等库进行运行时验证
  • XSS 防护 - 避免使用 dangerouslySetInnerHTML
  • 依赖安全 - 定期检查依赖漏洞

类型安全边界

明确区分可信任与不可信任的数据：

  • API 响应视为不可信任，使用 unknown 类型
  • 用户输入永远不可信任，需要验证后才能使用
  • 验证后的数据可以安全使用
  • 使用品牌类型标记已验证数据

使用技巧

unknown 类型 - API 响应使用 unknown

3
