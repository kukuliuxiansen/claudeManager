TypeScript 测试规范

定义单元测试、集成测试、E2E 测试和 Mock 策略

类型: 语言规则

实现效果

TypeScript 测试规范确保类型安全测试和完整的测试覆盖。

  • 单元测试 - 使用 Jest 或 Vitest 测试独立函数和组件
  • 集成测试 - 测试 API 端点和数据库交互
  • E2E 测试 - 使用 Playwright 或 Cypress 测试用户流程
  • Mock 策略 - 使用类型安全的 mock 和 spy

使用技巧

1

Jest/Vitest - 使用现代测试框架

2

类型安全 Mock - 使用 ts-mockito 或 jest.mock

3

Playwright E2E - 端到端测试首选

4

覆盖率报告 - 保持 80%+ 覆盖率
