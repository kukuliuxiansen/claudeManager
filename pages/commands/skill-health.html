技能健康命令

显示技能健康状况仪表盘和失败模式分析

类型: Command类

实现效果

技能健康命令运行 skills-health CLI 工具的仪表盘模式，显示技能的成功率、失败模式和变更提案等健康指标。

  • 成功率图表

    显示过去30天每日成功率趋势

  • 失败模式聚类

    水平条形图显示失败原因分布

  • 待处理变更

    显示等待审核的变更提案

  • 版本历史

    技能版本快照时间线

    • 核心原理

    技能健康命令的核心原理是运行 skills-health CLI 并解析 JSON 输出。通过执行 nodes scripts/skills-health.js 脚本获取技能使用数据，包括成功率、失败模式和变更提案。

    数据来源:

    • 技能日志: 记录每次技能调用的成功/失败状态
    • 失败聚类: 相同失败原因的技能自动归类
    • 版本快照: 每次技能修改时创建版本快照

    健康阈值:

    • 80%+ 成功率 - 健康
    • 60-80% 成功率 - 警告
    • <60% 成功率 - 需要关注

    使用方法

    当需要查看技能健康状况或诊断问题技能时使用此命令。

    触发场景与调用 
    // 完整仪表盘视图
/skill-health

// 仅查看失败聚类面板
/skill-health --panel failures

// 输出机器可读的 JSON
/skill-health --json

    使用技巧

    1

    定期检查 - 定期运行以监控技能健康状况

    2

    关注低成功率 - 成功率低于60%的技能需要审查

    3

    查看失败模式 - 使用 --panel failures 查看失败详情

    4

    处理待处理变更 - 审核并合并待处理的变更提案
