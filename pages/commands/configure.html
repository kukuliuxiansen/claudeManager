配置命令

启用或禁用 Hookify 规则以自定义 Claude Code 行为

类型: Command类

实现效果

配置命令允许用户选择性地启用或禁用 Hookify 规则，从而自定义 Claude Code 的自动行为。

  • 发现规则
    扫描所有 .claude/hookify.*.local.md 规则文件
  • 选择性激活
    让用户选择要启用或禁用的规则
  • 实时生效
    更改立即生效，无需重启

    • 核心原理

    配置系统通过 YAML frontmatter 管理规则状态：

    1. 发现规则 - 使用 Glob 工具扫描 .claude/hookify.*.local.md 文件
    2. 读取状态 - 提取每个规则文件的 YAML frontmatter 中的 enabled 字段
    3. 用户选择 - 询问用户选择要切换的规则
    4. 状态切换 - 更改 enabled: true/false 字段
    5. 确认结果 - 显示更改摘要

    文件格式：

    • enabled: true - 规则启用
    • enabled: false - 规则禁用

    使用方法

    触发方式 
    /configure

    运行后，系统将：

    执行流程 
    1. 扫描 .claude/hookify.*.local.md 文件
2. 读取每个规则的 enabled 状态
3. 询问用户选择要切换的规则
4. 更新 selected 规则的 enabled 状态
5. 显示变更摘要

    使用技巧

    1
    定期审查
    定期检查启用的规则保持最佳配置
    2
    渐进启用
    首次使用时逐个启用规则
    3
    禁用干扰
    禁用干扰性规则以保持专注
    4
    项目配置
    不同项目可有不同的规则组合

    随时运行 /configure 快速切换
    6

    使用 /hookify:list 查看所有规则
