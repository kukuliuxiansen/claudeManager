设置包管理器命令

在 macOS/Linux 系统上自动检测和配置 JavaScript 包管理器

类型: Command类

实现效果

设置包管理器命令在项目中自动检测正确的 JavaScript 包管理器（pnpm、bun、yarn 或 npm），并支持全局和项目级别的配置。

  • 自动检测
    按优先级顺序检测包管理器：环境变量 > 项目配置 > package.json > 锁文件
  • 多级配置
    支持全局（~/.claude）和项目级（.claude/）配置文件
  • 环境变量覆盖
    CLAUDE_PACKAGE_MANAGER 环境变量可覆盖所有其他设置

    • 核心原理

    该命令按以下优先级顺序检测包管理器：

    1. 环境变量：检查 CLAUDE_PACKAGE_MANAGER
    2. 项目配置：读取 .claude/package-manager.json
    3. package.json：检查 packageManager 字段
    4. 锁文件：检测 package-lock.json、yarn.lock、pnpm-lock.yaml 或 bun.lockb
    5. 全局配置：读取 ~/.claude/package-manager.json
    6. 回退：自动选择第一个可用的包管理器（pnpm > bun > yarn > npm）

    使用方法

    触发场景与调用 
    # 检测当前包管理器
node scripts/setup-package-manager.js --detect

# 设置全局包管理器
node scripts/setup-package-manager.js --global pnpm

# 设置项目包管理器
node scripts/setup-package-manager.js --project bun

# 列出当前配置
node scripts/setup-package-manager.js --list

    使用技巧

    1
    环境变量优先
    设置 CLAUDE_PACKAGE_MANAGER 可即时覆盖所有配置
    2
    项目配置
    在 .claude/package-manager.json 中为特定项目设置
    3
    检查锁文件
    命令会检测现有锁文件以保持一致性
    4
    回退机制
    如果没有配置，自动选择 pnpm、bun、yarn、npm 中的第一个

    输出示例

    Package Manager Output 
    ### Detection Priority
1. Environment variable: CLAUDE_PACKAGE_MANAGER
2. Project config: .claude/package-manager.json
3. package.json: packageManager field
4. Lock file: package-lock.json, yarn.lock, pnpm-lock.yaml, bun.lockb
5. Global config: ~/.claude/package-manager.json
6. Fallback: First available (pnpm > bun > yarn > npm)

### Run The Detection
$ node scripts/setup-package-manager.js --detect
Current Package Manager: pnpm
Detected from: .claude/package-manager.json

### Configuration Files
# Global Configuration
// ~/.claude/package-manager.json
{
  "packageManager": "pnpm"
}

# Project Configuration
// .claude/package-manager.json
{
  "packageManager": "bun"
}

# package.json
{
  "packageManager": "pnpm@8.6.0"
}

### Environment Variable
$env:CLAUDE_PACKAGE_MANAGER = "pnpm"
export CLAUDE_PACKAGE_MANAGER=pnpm

### Supported Package Managers
| Manager | Lock File | Global Config |
|---------|-----------|---------------|
| pnpm    | pnpm-lock.yaml | ~/.claude/package-manager.json |
| bun     | bun.lockb | ~/.claude/package-manager.json |
| yarn    | yarn.lock | ~/.claude/package-manager.json |
| npm     | package-lock.json | ~/.claude/package-manager.json |
