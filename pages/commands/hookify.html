Hookify创建钩子

从对话分析或明确指令创建钩子规则

类型: Command类

实现效果

使用 hookify 插件从对话行为创建自定义钩子规则，防止不期望的行为：

  • 三种创建方式
    分析对话 / 明确指令 / 手动创建规则文件
  • 正则表达式匹配
    使用 Python 正则表达式定义触发模式
  • 即刻生效
    规则创建后立即生效，无需重启

    • 核心原理 

    
    Hookify 工作原理
    

    
钩子系统架构:

1. 钩子事件类型
├─ PreToolUse    - 工具执行前
├─ PostToolUse   - 工具执行后
├─ Stop          - 会话停止时
├─ UserPromptSubmit - 用户提交提示时
└─ all           - 所有事件

2. 配置文件位置
├─ .claude/hookify.{rule-name}.local.md
└─ 位于项目根目录的 .claude/ 目录

3. 规则文件格式
```markdown
name: rule-name
enabled: true
event: bash|file|stop|prompt|all
pattern: regex_pattern
action: warn|block
警告/阻止消息...
```

4. 钩子触发流程
├─ 用户提交动作
├─ 钩子读取规则文件
├─ 检查模式匹配
├─ 执行 action (warn/block)
└─ 显示消息或阻止操作

5. 动态加载
├─ 无需重启 Claude Code
├─ 文件修改立即生效
└─ 下次工具使用时读取

    使用方法

    触发场景与调用 
    # 创建钩子规则时触发

"/hookify [instructions]"
格式: 从对话或指令创建钩子规则
要求: 运行 /hookify 开始创建

示例用法：
1. /hookify 不要在生产文件中使用 console.log
2. /hookify 提醒我避免使用 chmod 777
3. /hookify 在停止前确认已运行测试

可选参数:
无参数 - 分析最近对话，找出不期望的行为

使用方式:
1. 明确指令 - 直接描述想要防止的行为
2. 对话分析 - 使用无参数 /hookify，系统自动分析对话
3. 手动创建 - 在 .claude/ 目录创建 .local.md 文件

规则文件格式:
name: warn-dangerous-rm
enabled: true
event: bash
pattern: rm\s+-rf
⚠️ **危险的 rm 命令检测！**

    使用技巧

    1
    使用 `/hookify:list` 查看所有现有规则
    2
    使用 `/hookify:configure` 交互式启用/禁用规则
    3
    正则表达式测试：`python3 -c "import re; print(re.search(r'pattern', 'test'))"`
    4
    action 设置为 `block` 可阻止危险操作，`warn` 只显示警告
    5
    规则文件添加到 .gitignore，避免提交到共享仓库
    6
    使用/\hookify:help 查看所有 hookify 命令

    Hookify 规则创建

步骤 1: 分析请求
用户要求: 不要在生产文件中使用 console.log

步骤 2: 确定参数
- 事件类型: file (Edits/Writes)
- 匹配模式: console\.log\(
- 动作: warn (警告但允许)
- 文件位置: .claude/hookify.warn-console-log.local.md

步骤 3: 询问确认
问题: 这应该阻止操作还是只显示警告？
选项:
  - "只警告" (action: warn - 显示消息但允许)
  - "阻止操作" (action: block - 防止执行)

步骤 4: 创建规则文件

已创建规则文件：
.clause/hookify.warn-console-log.local.md

内容：
name: warn-console-log
enabled: true
event: file
pattern: console\.log\(
⚠️ **console.log 检测**
您在文件中使用了 console.log。

请记得在提交前删除调试日志。

注意：规则文件已立即生效！
尝试使用 console.log 触发它。

其他可用命令：
- /hookify:list - 列出所有规则
- /hookify:configure - 配置规则
- /hookify:help - 帮助文档

免费提示：
规则文件位于：{cwd}/.claude/hookify.warn-console-log.local.md
