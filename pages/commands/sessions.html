会话管理命令

管理会话别名、列出会话、加载和显示会话信息

类型: Command类

实现效果

会话管理命令提供了一套完整的会话管理功能，包括列出所有会话、创建易记别名、加载历史会话内容以及查看会话统计信息。

  • 会话列表
    显示所有会话及其元数据，支持过滤和分页
  • 别名管理
    创建、删除和列出会话别名以简化访问
  • 会话统计
    显示行数、完成项、进行中项和文件大小

    • 核心原理

    该命令通过以下子命令管理会话：

    1. /sessions list [选项]：列出所有会话，可按日期或 ID 搜索过滤
    2. /sessions load <id|alias>：加载并显示会话内容
    3. /sessions alias <id> <name>：为会话创建别名
    4. /sessions alias --remove <name>：删除别名
    5. /sessions info <id|alias>：显示会话详细统计信息
    6. /sessions aliases：列出所有会话别名

    使用方法

    触发场景与调用 
    /sessions                                # 列出所有会话（默认）
/sessions list                           # 同上
/sessions list --limit 10                # 显示 10 个会话
/sessions list --date 2024-01-15         # 按日期过滤
/sessions list --search abc              # 搜索会话 ID
/sessions load <id|alias>                # 加载会话
/sessions alias <id> <name>              # 创建别名
/sessions alias --remove <name>          # 删除别名
/sessions unalias <name>                 # 同上
/sessions info <id|alias>                # 显示会话信息
/sessions aliases                          # 列出所有别名
/sessions help                             # 显示帮助

    使用技巧

    1
    使用短 ID
    通常前 4-8 个字符就足够唯一识别
    2
    为常用会话创建别名
    如 "today"、"web-project" 等
    3
    检查会话时间
    超过 7 天的会话可能需要验证上下文
    4
    查看分支和工作树
    会话元数据包含项目信息便于识别
