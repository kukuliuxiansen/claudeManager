Claw 命令

启动 NanoClaw REPL 会话，支持多技能和会话管理

类型: Command类

实现效果

Claw 命令启动一个轻量级的 NanoClaw REPL 会话，提供独立的对话环境，支持多技能热加载和会话管理功能。

  • 零依赖 REPL
    NanoClaw 保持零依赖，快速启动
  • 会话管理
    支持创建、列举和览会话
  • 技能热加载
    运行时动态加载技能到上下文中

    • 核心原理

    NanoClaw 是一个轻量级的 REPL 工具，提供完整的会话管理功能：

    1. REPL 循环 - 读取用户输入，执行命令，打印结果
    2. 会话存储 - 会话以 Markdown 格式存储在 ~/.claude/claw/<session>.md
    3. 技能热加载 - 运行时读取技能文件并注入到上下文中
    4. 就会话压缩 - 保留最近轮次，优化上下文宽度

    会话存储位置：

    • 默认路径: ~/.claude/claw/<session>.md
    • 环境变量 CLAW_SESSION 可指定会话名称

    使用方法

    触发方式 
    node scripts/claw.js
# 或
npm run claw

    环境变量配置：

    环境变量 
    CLAW_SESSION   # 会话名称（字母数字和连字符），默认 'default'
CLAW_SKILLS    # 启动时加载的技能列表，逗号分隔
CLAW_MODEL     # 默认模型（sonnet/haikuopus），默认 'sonnet'

    使用技巧

    1
    命名会话
    使用描述性会话名称便于识别
    2
    技能预加载
    通过 CLAW_SKILLS 环境变量启动时加载技能
    3
    会话压缩
    长时间会话运行 /compact 导航
    4
    导出会话
    使用 /export 命令导出为 Markdown 或 JSON
    5
    搜索历史
    使用 /search 查询跨会话语句
    6
    分支会话
    使用 /branch 创建当前会话的分支

    输出示例

    REPL 命令列表 
    /help                          显示帮助信息
/clear                         清除当前会话历史
/history                       打印完整对话历史
/sessions                      列出保存的会话
/model [name]                  显示/设置模型
/load <skill-name>             热加载技能到上下文
/branch <session-name>         分支当前会话
/search <query>                在会话中搜索查询
/compact                       压缩旧轮次，保持近期上下文
/export <md|json|txt> [path]   导出会话
/metrics                       显示会话指标
exit                           退出
