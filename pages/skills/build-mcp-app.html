构建MCP应用

构建带交互式UI组件的MCP应用，在聊天界面中渲染表单、选择器、图表等组件

类型: MCP应用

实现效果

构建带交互式UI的MCP应用，在Claude和ChatGPT中渲染交互组件。

  • UI资源声明
    通过_meta.ui.resourceUri声明UI资源，工具返回数据而非HTML，资源单独注册服务HTML
  • App类运行时
    使用@modelcontextprotocol/ext-apps的App类实现持久双向连接，接收工具结果并发送用户操作
  • 两种部署形态
    远程HTTP服务器或MCPB打包本地服务器，后者适用于驱动本地应用场景
  • 优雅降级
    不支持UI的host自动忽略_meta.ui，渲染工具的文本内容，自动降级

    • 核心原理

    何时使用Widget

    工具需要结构化输入、用户从大列表选择、破坏性操作确认、空间/视觉输出、长时间运行监控时使用Widget。否则返回文本即可。

    Widget vs Elicitation

    简单确认、短枚举选择、平表单用Elicitation（零UI代码）。大列表搜索、视觉预览、图表、实时更新用Widget。

    架构约束

    Widget运行在iframe沙箱中，无法访问host页面的DOM/cookies/storage，网络请求受CSP限制，需通过callServerTool路由。

    设计原则

    一个工具一个Widget，工具描述必须提及Widget，Widget在运行时可选，只读工具不要阻塞Widget结果。

    使用方法

    Widget工具注册 
    # 安装
npm install @modelcontextprotocol/sdk @modelcontextprotocol/ext-apps zod express

# 工具注册 - 返回数据，声明UI资源
registerAppTool(server, "pick_contact", {
  description: "Open an interactive contact picker",
  inputSchema: { filter: z.string().optional() },
  _meta: { ui: { resourceUri: "ui://widgets/picker.html" } },
}, async ({ filter }) => {
  const contacts = await db.contacts.search(filter);
  return { content: [{ type: "text", text: JSON.stringify(contacts) }] };
});

# 资源注册 - 服务HTML
registerAppResource(server, "Contact Picker", "ui://widgets/picker.html", {},
  async () => ({
    contents: [{
      uri: "ui://widgets/picker.html",
      mimeType: RESOURCE_MIME_TYPE,  // "text/html;profile=mcp-app"
      text: pickerHtml
    }],
  }),
);

# Widget运行时 - App类
const app = new App({ name: "ContactPicker", version: "1.0.0" }, {});
app.ontoolresult = ({ content }) => render(JSON.parse(content[0].text));
await app.connect();
app.sendMessage({ role: "user", content: [{ type: "text", text: "Selected" }] });

    使用技巧

    1
    必须内联ext-apps包到Widget HTML中，iframe的CSP会阻止CDN脚本加载导致Widget空白
    2
    一个工具一个Widget，Claude对清晰的工具边界推理效果更好
    3
    使用app.openLink打开外部链接，window.open和target="_blank"被沙箱阻止
    4
    遵循host主题，通过app.getHostContext()?.theme切换暗色模式，禁用multiply混合模式

    App类API 
    # App类方法
| Method                | Direction      | Use for                        |
|-----------------------|----------------|--------------------------------|
| onttoolresult = fn    | Host -> widget | 接收工具返回值                  |
| onttoolinput = fn     | Host -> widget | 接收工具输入参数                |
| sendMessage({...})    | Widget -> host | 注入消息到对话                  |
| updateModelContext()  | Widget -> host | 静默更新上下文                  |
| callServerTool()      | Widget -> server| 调用服务器其他工具              |
| openLink({url})       | Widget -> host | 在新标签打开URL                 |
| getHostContext()      | Host -> widget | 获取主题、locale等上下文        |

# Widget限制
- 无法访问host页面的DOM、cookies、storage
- 网络请求受CSP限制，需通过callServerTool路由
- 无法打开弹窗，需使用openLink
- 远程图片不可靠，需服务端内联为data: URL
