writing-hookify-rules

规则转钩子技能，将规则转换为可执行的钩子

类型: Plugin Skill · 来源: hookify

实现效果

规则转钩子技能，将自然语言规则转换为可执行的Claude Code钩子脚本。支持PreToolUse、PostToolUse、Stop等钩子类型。

  • 规则解析

    解析自然语言规则，提取条件和动作

  • 钩子生成

    生成可执行的钩子脚本代码

    • 使用方法

    // 转换规则为钩子
/hookify "禁止在测试文件中使用console.log"

// 生成钩子脚本
hookify generate --rule "no-console-in-tests" --type PostToolUse
