exa-search

Exa AI神经搜索引擎集成，提供网页搜索、代码查找、公司研究和深度研究能力

类型: API/Tools Skill

实现效果

exa-search 是 Exa AI 神经搜索引擎的 MCP 集成技能，提供高质量的网页搜索、代码示例查找、公司研究和 AI 驱动的深度研究能力。

  • 网页搜索

    web_search_exa 和 web_search_advanced_exa 支持过滤域名和日期的高级搜索

  • 代码上下文

    get_code_context_exa 从 GitHub、Stack Overflow、文档站点查找代码示例

  • 公司研究

    company_research_exa 快速研究公司背景、新闻和市场信息

  • 深度研究

    deep_researcher_start/check 异步 AI 研究代理，综合分析复杂主题

    • 核心原理

    Exa 使用神经搜索引擎而非传统关键词搜索，能够理解查询意图并返回语义相关的结果。通过 MCP 服务器集成，Claude Code 可以直接调用 Exa 的搜索能力。

    MCP 配置要求:

    • 在 ~/.claude.json 中配置 exa-web-search MCP 服务器
    • 需要 Exa API Key (从 exa.ai 获取)
    • 可指定启用的工具列表或使用默认集

    核心工具:

    • web_search_exa: 通用网页搜索，获取当前信息和新闻
    • web_search_advanced_exa: 带域名和日期过滤的高级搜索
    • get_code_context_exa: 查找代码示例和技术文档
    • company_research_exa: 公司背景和市场情报研究
    • crawling_exa: 提取指定 URL 的完整内容
    • deep_researcher: AI 驱动的深度研究代理

    使用方法

    exa-search 适用于需要当前网络信息、代码示例、公司研究或深度分析的场景。

    触发场景与调用 
    // 主动触发条件
1. 用户需要当前网络信息或新闻
2. 搜索代码示例、API文档或技术参考
3. 研究公司、竞争对手或市场参与者
4. 查找专业资料或领域专家
5. 运行开发任务的背景研究

// 快速查找
web_search_exa(query: "Node.js 22 new features", numResults: 3)

// 代码研究
get_code_context_exa(query: "Rust error handling patterns", tokensNum: 3000)

// 公司调研
company_research_exa(companyName: "Anthropic", numResults: 5)
web_search_advanced_exa(
  query: "Vercel funding valuation 2026",
  numResults: 3
)

// 技术深度研究
deep_researcher_start(query: "WebAssembly component model status")
// ... 其他工作 ...
deep_researcher_check(researchId: "<id>")

// 带过滤的高级搜索
web_search_advanced_exa(
  query: "React Server Components best practices",
  numResults: 5,
  includeDomains: ["github.com", "react.dev"],
  startPublishedDate: "2025-01-01"
)

    使用技巧

    1

    选择合适工具 - 宽泛查询用 web_search_exa，过滤查询用 advanced

    2

    控制 tokens - 聚焦代码用低 tokens (1000-2000)，全面上下文用高 (5000+)

    3

    组合研究 - company_research + web_search_advanced 做全面公司分析

    4

    提取内容 - 用 crawling_exa 获取搜索结果中的特定 URL 内容

    5

    深度研究 - deep_researcher 适合需要 AI 综合分析的复杂主题
