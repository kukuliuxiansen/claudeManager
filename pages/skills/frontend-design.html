frontend-design

创建独特的、生产级前端界面，避免通用的AI生成美学，追求高设计质量

类型: Pattern Skill

实现效果

frontend-design 技能用于创建独特的、生产级前端界面，强调设计质量和创意选择，避免通用 AI 生成的"千篇一律"美学。

  • 独特美学方向

    选择并执行明确的设计方向：极简主义、最大主义、复古未来、有机自然等

  • 字体与排版

    选择独特、有个性的字体，避免 Arial、Inter 等通用字体

  • 动态与交互

    使用动画和微交互，CSS 动画优先，React 用 Motion 库

  • 视觉细节

    渐变网格、噪点纹理、几何图案、透明层叠、装饰边框等

    • 核心原理

    前端设计的核心是明确概念方向并精确执行。大胆的最大主义和精致的极简主义都可行，关键是有意图性而非强度。

    设计思考框架:

    • 目的: 这个界面解决什么问题？谁在使用它？
    • 基调: 选择一个极端方向：极简、最大主义、复古未来、有机自然等
    • 约束: 技术要求（框架、性能、可访问性）
    • 差异化: 什么让它难忘？那个让人记住的一点是什么？

    避免的通用美学:

    • 过度使用的字体：Inter, Roboto, Arial, Space Grotesk
    • 陈词滥调的配色：白底上的紫色渐变
    • 可预测的布局和组件模式
    • 缺乏上下文特色的千篇一律设计

    使用方法

    frontend-design 适用于构建网页组件、页面或应用程序时，要求创造独特、精致的设计。

    触发场景与设计流程 
    // 主动触发条件
1. 用户要求构建网页组件、页面或应用
2. 需要创建独特的视觉设计
3. 避免通用的 AI 生成美学

// 设计思考流程
1. 理解上下文和目的
2. 选择大胆的美学方向
3. 实现可工作的代码 (HTML/CSS/JS, React, Vue等)
4. 确保生产级质量、视觉冲击力、连贯的美学观点

// 字体选择原则
- 选择独特、有个性、有趣的字体
- 避免 Arial、Inter、Roboto、系统字体
- 配对一个独特的展示字体和一个精致的正文字体

// 动画原则
- HTML 优先使用 CSS-only 解决方案
- React 使用 Motion 库
- 关注高影响时刻：页面加载时的错落展示
- 使用滚动触发和惊喜悬停状态

// 空间构图
- 意想不到的布局、不对称、重叠、对角线流动
- 打破网格的元素、慷慨的留白或受控的密度

// 背景与视觉细节
- 创建氛围和深度，而非默认纯色
- 渐变网格、噪点纹理、几何图案
- 层叠透明、戏剧性阴影、装饰边框
- 自定义光标、颗粒覆盖

    使用技巧

    1

    大胆选择 - 最大主义和极简主义都有效，关键是有意图性

    2

    匹配复杂度 - 复杂设计需要精心代码，简约需要精确细节

    3

    独特字体 - 避免通用选择，每个设计都应不同

    4

    主题变化 - 在明暗主题、字体、美学之间变化

    5

    上下文设计 - 做出真正为上下文设计的意外选择

## 设计决策

### 字体选择
- 展示字体: "Playfair Display" (优雅衬线)
- 正文字体: "DM Sans" (现代无衬线)

### 配色方案
--primary: #1a1a2e (深紫)
--accent: #d4af37 (金色)
--surface: #16213e (深蓝)
--text: #e8e8e8

### 动画策略
- 页面加载: 错落的几何图形展开
- 悬停: 金色边框渐显
- 滚动: 视差几何图案

### 视觉细节
- 几何图案背景 (CSS 渐变 + clip-path)
- 金色装饰边框 (伪元素)
- 噪点纹理覆盖 (SVG filter)
- 自定义光标 (圆形金色环)
