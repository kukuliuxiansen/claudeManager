frontend-patterns

前端开发模式和组件设计模式，涵盖状态管理、性能优化、表单处理和可访问性

类型: Pattern Skill

实现效果

frontend-patterns 是 React、Next.js 和高性能用户界面的现代前端开发模式指南，帮助构建可维护、高性能的 UI。

  • 组件模式

    组合优于继承、复合组件、Render Props 模式

  • 自定义 Hooks

    状态管理、异步数据获取、防抖、窗口大小等常用 Hooks

  • 性能优化

    记忆化、代码分割、懒加载、长列表虚拟化

  • 可访问性

    键盘导航、焦点管理、ARIA 属性、屏幕阅读器支持

    • 核心原理

    现代前端开发的核心是组合优于继承，通过组件组合、自定义 Hooks 和性能优化模式构建可维护的应用。

    组件设计原则:

    • 组合模式: 使用 children 和组合而非继承
    • 复合组件: 通过 Context 共享状态的组件族
    • Render Props: 通过函数 prop 实现逻辑复用

    性能优化策略:

    • useMemo: 缓存昂贵计算
    • useCallback: 缓存传递给子组件的函数
    • React.memo: 跳过纯组件的重渲染
    • 懒加载: React.lazy + Suspense 代码分割
    • 虚拟化: @tanstack/react-virtual 处理长列表

    使用方法

    frontend-patterns 适用于构建 React 组件、管理状态、实现数据获取、优化性能、处理表单等场景。

// 组件组合模式
interface CardProps {
  children: React.ReactNode
  variant?: 'default' | 'outlined'
}

export function Card({ children, variant = 'default' }: CardProps) {
  return <div className={`card card-${variant}`}>{children}</div>
}

// 自定义 Hook - 防抖
export function useDebounce<T>(value: T, delay: number): T {
  const [debouncedValue, setDebouncedValue] = useState<T>(value)

  useEffect(() => {
    const handler = setTimeout(() => {
      setDebouncedValue(value)
    }, delay)
    return () => clearTimeout(handler)
  }, [value, delay])

  return debouncedValue
}

// 性能优化 - 虚拟化长列表
import { useVirtualizer } from '@tanstack/react-virtual'

export function VirtualList({ items }: { items: Item[] }) {
  const parentRef = useRef<HTMLDivElement>(null)

  const virtualizer = useVirtualizer({
    count: items.length,
    getScrollElement: () => parentRef.current,
    estimateSize: () => 100,
    overscan: 5
  })

}

## Applied Patterns

### 1. Compound Components
- Tabs with shared context
- Modal with overlay + content

### 2. Custom Hooks
- useDebounce for search
- useQuery for data fetching

### 3. Performance
- React.memo for 15 components
- Lazy loading 3 heavy components
- Virtualization for 1000+ item lists

### 4. Accessibility
- Keyboard navigation for dropdowns
- Focus management for modals
- ARIA labels for all interactive elements
