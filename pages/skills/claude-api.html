claude-api

Anthropic Claude API 模式，支持 Python 和 TypeScript SDK

类型: API Skill

实现效果

claude-api Skill 提供完整的 Claude API 使用指南，涵盖 Messages API、Tool Use、Vision、Extended Thinking 等核心功能，帮助开发者快速构建 AI 应用。

  • Messages API

    支持基本消息、流式响应、系统提示等核心对话功能

  • Tool Use

    定义工具让 Claude 调用，实现 Agent 工作流和多步骤任务

  • Vision

    发送图片进行分析，支持多模态理解和描述

  • Extended Thinking

    复杂推理任务，预留思考预算进行深度分析

    • 核心原理

    Claude API 基于 Anthropic 的 Messages API，支持多种模型选择和高级功能。核心模型包括 Opus 4.1（复杂推理）、Sonnet 4（平衡开发）、Haiku 3.5（快速响应）。

    模型选择策略:

    • Opus 4.1: 复杂推理、架构设计、研究分析
    • Sonnet 4: 平衡开发、大多数开发任务
    • Haiku 3.5: 快速响应、高并发、成本敏感场景

    成本优化策略:

    • Prompt Caching: 缓存可达 90% 成本节省
    • Batches API: 批量处理节省 50%
    • Haiku 替代 Sonnet: 简单任务节省约 75%

    使用方法

    claude-api Skill 在构建 Claude API 应用或使用 Anthropic SDK 时触发：

    Python SDK 基础使用 
    # 安装 SDK
pip install anthropic

# 基本消息
import anthropic

client = anthropic.Anthropic()  # 自动读取 ANTHROPIC_API_KEY

message = client.messages.create(
    model="claude-sonnet-4-0",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "解释 async/await"}
    ]
)
print(message.content[0].text)

# 流式响应
with client.messages.stream(
    model="claude-sonnet-4-0",
    max_tokens=1024,
    messages=[{"role": "user", "content": "写一首诗"}]
) as stream:
    for text in stream.text_stream:
        print(text, end="", flush=True)

# Tool Use 示例
tools = [{
    "name": "get_weather",
    "description": "获取天气信息",
    "input_schema": {
        "type": "object",
        "properties": {
            "location": {"type": "string"}
        },
        "required": ["location"]
    }
}]

message = client.messages.create(
    model="claude-sonnet-4-0",
    max_tokens=1024,
    tools=tools,
    messages=[{"role": "user", "content": "北京天气如何？"}]
)

# 处理 tool_use 响应
for block in message.content:
    if block.type == "tool_use":
        result = get_weather(**block.input)
        # 发送结果回 Claude

    使用技巧

    1

    Prompt Caching - 缓存重复的系统提示，节省 API 成本

    2

    环境变量 - 使用 ANTHROPIC_API_KEY，切勿硬编码密钥

    3

    错误处理 - 处理 RateLimitError、APIConnectionError 等异常

    4

    并行执行 - 使用 Promise.all 并行调用多个独立请求

Message(
    id='msg_01ABC...',
    type='message',
    role='assistant',
    content=[
        TextBlock(
            type='text',
            text='这是 Claude 的回复内容...'
        )
    ],
    model='claude-sonnet-4-0',
    stop_reason='end_turn',
    usage={
        'input_tokens': 25,
        'output_tokens': 150,
        'cache_read_input_tokens': 0,
        'cache_creation_input_tokens': 0
    }
)

# Tool Use 响应
Message(
    content=[

            type='tool_use',

            input={'location': '北京'}
        )
    ]
)
