claude-md-improver

CLAUDE.md 文件审计和改进，确保 Claude Code 拥有最佳项目上下文

类型: Documentation Skill

实现效果

claude-md-improver Skill 审计和改进 CLAUDE.md 文件，确保 Claude Code 拥有最佳项目上下文。扫描所有 CLAUDE.md 文件，评估质量，输出质量报告，然后进行针对性更新。

  • Discovery

    扫描项目中所有 CLAUDE.md 文件，包括根目录、子目录、monorepo 包级别

  • Quality Assessment

    按质量标准评估每个文件，包括命令、架构清晰度、可操作性等

  • Quality Report

    输出详细质量报告，包含评分、问题列表和推荐改进

  • Targeted Updates

    用户确认后，应用针对性更新，添加缺失的关键信息

    • 核心原理

    CLAUDE.md 是 Claude Code 的项目上下文文件，包含项目特定的命令、架构、gotcha 和工作流。好的 CLAUDE.md 应简洁、可操作、项目特定，而非通用建议。

    质量评估维度:

    • Commands/workflows (高权重): 构建/测试/部署命令是否存在
    • Architecture clarity (高权重): Claude 能否理解代码库结构
    • Non-obvious patterns (中权重): gotcha 和 quirks 是否记录
    • Conciseness (中权重): 无冗长解释或显而易见的信息
    • Currency (高权重): 反映当前代码库状态
    • Actionability (高权重): 指令可执行，非模糊描述

    评分等级:

    • A (90-100): 全面、当前、可操作
    • B (70-89): 良好覆盖，小差距
    • C (50-69): 基础信息，缺失关键部分
    • D (30-49): 稀疏或过时
    • F (0-29): 缺失或严重过时
