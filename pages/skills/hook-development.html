hook-development

Claude Code插件Hook开发指南，提供事件驱动自动化脚本的完整开发流程

类型: API/Tools Skill

实现效果

hook-development 是 Claude Code 插件 Hook 开发的完整指南，涵盖 Prompt 驱动 Hook、命令 Hook 和各种事件类型的配置。

  • Prompt 驱动 Hook

    LLM 驱动的上下文感知验证，支持自然语言推理

  • 命令 Hook

    Bash 脚本确定性检查，用于快速验证和外部工具集成

  • 多事件类型

    PreToolUse、PostToolUse、Stop、SessionStart 等9种事件

  • 安全最佳实践

    输入验证、路径安全、变量引用、超时设置

    • 核心原理

    Hook 是响应 Claude Code 事件执行的事件驱动自动化脚本，用于验证操作、执行策略、添加上下文和集成外部工具。

    Hook 事件类型:

    • PreToolUse: 工具执行前 - 验证、修改
    • PostToolUse: 工具完成后 - 反馈、日志
    • Stop: 代理停止时 - 完整性检查
    • SessionStart: 会话开始 - 上下文加载
    • UserPromptSubmit: 用户提交 - 上下文、验证

    退出码:

    • 0: 成功 (stdout 显示在转录中)
    • 2: 阻塞错误 (stderr 反馈给 Claude)
    • 其他: 非阻塞错误

    使用方法

    hook-development 适用于创建 Hook、添加 PreToolUse/PostToolUse/Stop Hook、验证工具使用等场景。

    Hook 配置示例 
    // 主动触发条件
1. 创建 Hook、添加 PreToolUse/PostToolUse/Stop Hook
2. 验证工具使用、实现 Prompt 驱动 Hook
3. 使用 ${CLAUDE_PLUGIN_ROOT}、设置事件驱动自动化
4. 阻止危险命令、提及 Hook 事件

// Prompt 驱动 Hook (推荐)
{
  "PreToolUse": [
    {
      "matcher": "Write|Edit",
      "hooks": [
        {
          "type": "prompt",
          "prompt": "验证文件写入安全性。检查：系统路径、凭证、路径遍历、敏感内容。返回 'approve' 或 'deny'。"
        }
      ]
    }
  ]
}

// 命令 Hook
{
  "type": "command",
  "command": "bash ${CLAUDE_PLUGIN_ROOT}/scripts/validate.sh",
  "timeout": 60
}

// Stop Hook - 完整性验证
{
  "Stop": [
    {
      "matcher": "*",
      "hooks": [
        {
          "type": "prompt",
          "prompt": "验证任务完成：测试运行、构建成功、问题回答。返回 'approve' 停止或 'block' 继续并说明原因。"
        }
      ]
    }
  ]
}

// SessionStart Hook - 加载上下文
{
  "SessionStart": [
    {
      "matcher": "*",
      "hooks": [
        {
          "type": "command",
          "command": "bash ${CLAUDE_PLUGIN_ROOT}/scripts/load-context.sh",
          "timeout": 10
        }
      ]
    }
  ]
}

    使用技巧

    1

    Prompt 复杂逻辑 - 用 Prompt Hook 处理复杂推理

    2

    可移植路径 - 使用 ${CLAUDE_PLUGIN_ROOT}

    3

    验证输入 - 在命令 Hook 中验证所有输入

    4

    引用变量 - 引用所有 Bash 变量防止注入

    5

    设置超时 - 命令默认60s，Prompt默认30s

    6

    并行执行 - 所有匹配 Hook 并行运行

## Hook Events Summary
| Event | When | Use For |
|-------|------|---------|
| PreToolUse | Before tool | Validation, modification |
| PostToolUse | After tool | Feedback, logging |
| UserPromptSubmit | User input | Context, validation |
| Stop | Agent stopping | Completeness check |
| SubagentStop | Subagent done | Task validation |
| SessionStart | Session begins | Context loading |
