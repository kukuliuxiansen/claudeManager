browser-qa

浏览器QA测试技能，自动化端到端浏览器测试

类型: Testing Skill

实现效果

浏览器QA测试技能，通过Chrome DevTools MCP进行自动化端到端测试。验证用户界面、交互流程和跨浏览器兼容性。

  • 自动化交互测试

    模拟用户点击、输入、导航等交互行为

  • 截图与快照

    自动捕获测试过程截图，便于问题定位

    • 使用方法

    // 运行浏览器QA测试
/e2e --browser chrome

// Playwright测试示例
test('用户登录', async ({ page }) => {
  await page.goto('/login')
  await page.fill('[name="email"]', 'user@test.com')
  await page.click('button[type="submit"]')
})
