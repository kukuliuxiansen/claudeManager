Configure ECC

Everything Claude Code 交互式安装器 — 引导用户选择和安装 skills 和 rules

类型: Skill (ECC)

实现效果

一个交互式的、分步安装向导，用于 Everything Claude Code 项目。

  • 选择性安装 - 使用 AskUserQuestion 引导用户选择 skills/rules
  • 路径验证 - 安装后验证文件存在和路径引用
  • 优化选项 - 可选择优化已安装的文件
  • 依赖检查 - 验证 skills 之间的交叉引用

安装步骤

Step 0: 克隆 ECC 仓库到 /tmp
Step 1: 选择安装级别 (用户级/项目级/两者)
Step 2: 选择并安装 Skills
Step 3: 选择并安装 Rules
Step 4: 安装后验证
Step 5: 优化已安装文件 (可选)
Step 6: 安装摘要报告

使用技巧

1

通过插件 - /plugin install everything-claude-code

2

手动引导 - 复制 skill 到 ~/.claude/skills/

3

Core 推荐 - 新用户建议安装核心 skills

4

验证修复 - 安装后检查路径引用问题
