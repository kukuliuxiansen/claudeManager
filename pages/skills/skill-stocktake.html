skill-stocktake

技能盘点审计工具，评估所有技能质量并提供改进建议

类型: Skill Audit

实现效果

skill-stocktake Skill 使用AI整体判断审计所有Claude技能和命令，通过质量检查清单评估并提供改进建议。

  • 双模式扫描

    快速扫描（仅变更技能，5-10分钟）和完整盘点（所有技能，20-30分钟）两种模式

  • 五级裁决系统

    Keep（保留）、Improve（改进）、Update（更新）、Retire（退役）、Merge into X（合并）五种裁决结果

  • 质量检查清单

    检查内容重叠、MEMORY.md/CLAUDE.md重叠、技术参考新鲜度、使用频率四个维度

  • 结果缓存

    结果保存至results.json，支持增量评估和中断恢复

    • 核心原理

    技能盘点遵循"整体AI判断"原则。评估是整体AI判断而非数字评分，关注可操作性、范围适配、独特性和时效性。

    扫描范围:

    • 全局技能: ~/.claude/skills/ 目录
    • 项目技能: {cwd}/.claude/skills/ 目录（如存在）

    裁决标准:

    • 可操作性: 是否有代码示例、命令、步骤让人立即行动
    • 范围适配: 名称、触发条件、内容是否对齐
    • 独特性: 价值是否可被MEMORY.md/CLAUDE.md/其他技能替代
    • 时效性: 技术参考在当前环境是否有效

    使用方法

    skill-stocktake Skill 在需要审计技能质量时使用：

    运行命令 
    // 快速扫描（默认，仅评估变更技能）
/skill-stocktake

// 完整盘点（评估所有技能）
/skill-stocktake full

// 扫描阶段：运行inventory脚本
bash ~/.claude/skills/skill-stocktake/scripts/scan.sh

// 评估阶段：启动子代理评估
Agent(subagent_type="general-purpose", prompt="
  Evaluate the following skill inventory against the checklist.
  Return JSON for each skill:
  { \"verdict\": \"Keep|Improve|Update|Retire|Merge into [X]\", \"reason\": \"...\" }
")

// 结果保存
~/.claude/skills/skill-stocktake/results.json

    使用技巧

    1

    定期运行审计 - 安装新技能或大幅修改后运行完整盘点

    2

    关注Improve裁决 - 查看具体改进建议，决定是否执行

    3

    Retire需确认 - 退役/合并操作需要用户明确确认后才执行

    4

    检查MEMORY.md - 如果超过100行，建议压缩以减少上下文占用
