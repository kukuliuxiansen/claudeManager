Continuous Learning V2

Instinct-based 学习系统，支持项目范围隔离和置信度评分

类型: Learning Skill v2.1

实现效果

Continuous Learning V2 是基于 instinct 的学习系统，通过 hooks 观察会话，创建带有置信度评分的原子行为，并演化为 skills/commands/agents。

  • 100% 可靠观察

    PreToolUse/PostToolUse hooks 确保每次工具调用都被观察

  • 项目范围隔离

    React 模式留在 React 项目，Python 约定留在 Python 项目

  • 置信度评分

    0.3-0.9 加权评分，随观察增加或用户纠正而变化

  • 进化路径

    Instincts → cluster → skill/command/agent 完整进化链

    • 核心原理

    Instinct 是小型学习行为，包含触发条件、动作、置信度、领域标签和证据。项目范围隔离防止跨项目污染。

    v2.1 新特性:

    • Storage: 项目范围 (projects/<hash>/) + 全局
    • Detection: git remote URL / repo path 自动检测
    • Promotion: 项目 → 全局 (在 2+ 项目中出现)
    • Commands: 6 个命令 (status/evolve/export/import/promote/projects)

    项目检测优先级:

    • CLAUDE_PROJECT_DIR 环境变量 (最高)
    • git remote get-url origin (可移植 ID)
    • git rev-parse --show-toplevel (机器特定)
    • 全局回退 (无项目时)

    使用方法

    命令列表与 Instinct 示例 
    # Instinct 命令
/instinct-status     # 显示学习到的 instincts (项目 + 全局)
/evolve              # 聚类相关 instincts 为 skills/commands
/instinct-export     # 导出 instincts 到文件
/instinct-import     # 从他人导入 instincts
/promote             # 将项目 instincts 提升到全局
/projects            # 列出所有已知项目

# Instinct 示例 (YAML)
---
id: prefer-functional-style
trigger: "when writing new functions"
confidence: 0.7
domain: "code-style"
source: "session-observation"
scope: project
project_id: "a1b2c3d4e5f6"
project_name: "my-react-app"
---

# Prefer Functional Style
## Action
Use functional patterns over classes when appropriate.

## Evidence
- Observed 5 instances of functional pattern preference
- User corrected class-based approach on 2025-01-15

    使用技巧

    1

    使用 Hooks 观察而非 Skills - Hooks 100% 可靠触发，Skills 仅 50-80% 触发率

    2

    范围决策 - 语言/框架约定放项目，安全实践和工具工作流放全局

    3

    提升标准 - 同一 instinct 在 2+ 项目中出现且平均置信度 >= 0.8 可提升

    4

    隐私保护 - 观察数据保持本地，仅 instincts（模式）可导出
