### 步骤执行流程
1. 用户输入凭证 → Login组件
2. validateCredentials() 检查格式
3. API调用 authenticateUser()
4. backend验证密码哈希
5. 生成JWT token
6. 存储到数据库（users表）
7. 返回响应

### 关键组件职责
- UserService.ts: 用户CRUD操作
- AuthService.ts: JWT生成/验证
- AuthMiddleware.ts: API路由保护

### 架构洞察
- 模式: 中间件模式用于路由保护
- 层: 表示层 → 服务层 → 数据层
- 设计决策: 无状态认证使用JWT
- 依赖: jsonwebtoken, bcrypt
