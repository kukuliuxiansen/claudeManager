对话分析代理

分析对话模式并生成预防性规则的AI代理

类型: Agent类

实现效果

对话分析代理模式化 Claude Code 历史对话中的行为模式，检测可预防的问题并生成自动化规则。

  • 行为模式检测

    识别对话中反复出现的不良或危险行为模式

  • 严重级别分类

    按高、中、低严重性分类检测到的问题

  • 规则生成

    为高严重性问题生成正则表达式规则和预防提示

  • edge cases 处理

    区分不当行为、教学场景和偶然错误

    • 核心原理

    对话分析代理使用规则模式匹配方法，将对话历史中的工具使用模式与已知危险模式进行比较。

    分析原理:

    1. 模式提取
    • 从工具调用中提取关键信息
    • 识别高频行为模式
    • 关联用户响应和反馈
    2. 规则匹配
    • 将模式与危险模式数据库比较
    • 计算匹配分数和置信度
    • 应用严重级别分类
    3. 规则生成
    • 为检测到的模式创建正则表达式
    • 生成事件触发规则
    • 创建用户提示消息
    4. 安全边界
    • 区分实际问题和教学讨论
    • 标记偶然错误而非模式
    • 避免误报

    ### 问题 1: 危险的 rm 命令
**严重性**: 高
**工具**: Bash
**模式**: `rm\s+-rf`
**出现次数**: 3次
**上下文**: 在没有验证的情况下对/tmp目录使用rm -rf
**用户反应**: "请在执行rm命令时更加小心"
**建议规则:**
- 名称: warn-dangerous-rm
- 事件: bash
- 模式: rm\s+-rf
- 消息: "检测到危险的rm命令。请在执行前验证路径。"

### 问题 2: TypeScript中的console.log
**严重性**: 中
**工具**: Edit/Write
**模式**: `console\.log\(`
**出现次数**: 2次
**上下文**: 在生产TypeScript文件中添加console.log语句
**用户反应**: "不要在生产代码中使用console.log"
**建议规则:**
- 名称: warn-console-log
- 事件: file
- 模式: console\.log\(
- 消息: "检测到console.log。请使用适当的日志库。"

### 总结
发现 {N} 个可预防行为:
- {N} 个高严重性
- {N} 个中严重性
- {N} 个低严重性

建议为高和中严重性问题创建规则。

### 安全边界考虑
**用户正在讨论假设情况:**
- "如果我使用rm -rf会发生什么？"
- 不视为问题行为
**教学时刻:**
- "你不应该这样做: ..."
- 上下文表明是解释，而非问题
**一次性错误:**
- 单次出现，已修复
- 提及但标记为低优先级
**主观偏好:**
- "我更喜欢X而不是Y"
- 标记为低严重性，让用户决定

### 返回结果
以结构化格式提供分析。/hookify命令将使用此来:
1. 向用户显示发现
2. 询问要创建的规则
3. 生成 .local.md 配置文件
4. 将规则保存到 .claude 目录

    使用技巧

    1

    具体模式 - 避免过于宽泛，使用精确的正则表达式

    2

    真实示例 - 包含来自对话的实际代码示例

    3

    上下文说明 - 解释为什么每个问题有害

    4

    测试规则 - 验证规则不会产生误报

#### 检测到的模式
| 问题 | 模式 | 频率 | 严重性 |
|------|------|------|--------|
| 危险rm命令 | rm\s+-rf | 3次 | 高 |
| console.log泄漏 | console\.log\(| 2次 | 中 |

#### 推荐规则
1. **warn-dangerous-rm**
   - 匹配: rm\s+-rf
   - 消息: "危险的rm命令。请验证路径。"
   - 事件: bash

2. **warn-console-log**
   - 匹配: console\.log\(
   - 消息: "在生产代码中使用日志库。"
   - 事件: file

#### 生成的 .local.md
```yaml
rules:
  - name: warn-dangerous-rm
    event: bash
    pattern: rm\s+-rf
    message: "危险的rm命令。请验证路径。"
    severity: high
  - name: warn-console-log
    event: file
    pattern: console\.log\(
    message: "在生产代码中使用日志库。"
    severity: medium
```

#### 质量标准
- 模式具体，不宽泛
- 包含实际对话示例
- 解释问题重要性
- 提供可用的regex模式
- 不误报讨论内容
